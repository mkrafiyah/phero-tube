Question 1:Discuss the scope of var, let, and const.

Answer:
var: var is global and functional scoped.var variables can be updated and re-declared within its scope.
let: let is block scoped.let variables can be updated but not re-declared.
const: const is block scoped.const variables can neither be updated nor re-declared.



Question 2:Tell us the use cases of null and undefined.

Answer:
null:It means there is an empty value or a blank value.It can be assigned to a variable as a representation of no value.
undefined:Undefined means the variable has been declared, but its value has not been assigned.Ways to get undefined:
1.variable that is not initialized.
2.function with no returns.
3.parameter that is not passed.
4.property that does not exists on an objecet.



Question 3:What do you mean by REST API?

Answer:
An architectural style for an application program interface (API) that uses HTTP requests to access and use data is known as REST API.That data can be used to GET, PUT, POST and DELETE data types, which refers to the reading, updating, creating and deleting of operations concerning resources.A REST API is also known as RESTful API.
